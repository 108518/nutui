**NutUI 2.0计划**

- [ ] 跨平台，提交的Vue组件自动生成微信小程序组件
- [ ] 脚手架升级webpack 4.0，构建速度大幅提升
- [ ] 网站页面设计由设计师小姐姐把持，组件严格按照京东APP最新UI规范开发
- [ ] 统一的动效
- [ ] 更方便、更灵活、更优雅的按需引用方式
- [ ] 组件内建文案支持多语言（国际化）
- [ ] 支持自定义主题
- [ ] 基于MD文件自动生成文档
- [ ] 支持自动化测试和持续集成
- [ ] 官网将支持PWA
